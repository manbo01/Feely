# Feely - 감정 일기 앱

감정 변화를 기록하고 시각적으로 확인하는 iOS/Android 크로스플랫폼 앱입니다.

## 기술 스택

- Flutter (Dart)
- Provider (상태 관리)
- Hive (로컬 저장)
- flutter_local_notifications (알림)
- flutter_naver_map (네이버 지도)

## 실행 방법

1. [Flutter SDK](https://flutter.dev/docs/get-started/install) 설치
2. 프로젝트 루트에서:
   ```bash
   flutter pub get
   flutter run
   ```
3. iOS/Android 플랫폼 폴더가 불완전한 경우, 기존 `lib/`와 `pubspec.yaml`을 유지한 채로 플랫폼만 다시 생성하려면:
   ```bash
   flutter create . --project-name feely
   ```
   (이미 있는 Dart/설정 파일은 덮어쓰지 않도록 주의)

## 구현된 기능

- **감정 일기**: 날짜, 시간, 날씨(자동), 장소(자동), 감정 태그(사용자 추가 가능), 감정 강도(1~10, 그라데이션 바), 일기 내용, 사진 첨부, 저장
- **메인 화면**: 월별 캘린더, 날짜별 일기 목록(선택 시 일기 상세로 이동), 일기 있는 날 점으로 표시, 선택된 날 강조 표시
- **일기 상세**: 일기 열람, 수정 및 삭제
- **설정**: 테마 4종(라이트, 다크, 파랑, 초록), 알림 여부 및 시간 설정, 데이터 관리(미구현)
- **로컬 알림**: 지정한 시간에 푸시 알림 발송 (문구 랜덤: "지금 기분이 어때요?" 등 3종, 권한 필요)

## 장소 선택 지도 (네이버 지도)

- **네이버 지도**가 등록되어 있으며, 장소 자동 입력 시 지도에서 위치를 선택할 수 있습니다.

## 후순위(미구현)

- 감정 분석·요약
- 그룹 공유, 클라우드 백업
